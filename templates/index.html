Deckbox API (non-official)

Deckbox is a card collection manager.
With a smart and responsive user interface for adding cards, you can quickly create an Inventory, Tradelist and Wishlist.

Visit Deckbox

This application is an attempt to emulate a deckbox API.
It's based on a HTML parser that will retreive all public information from deckbox.
This is not the official deckbox API.
